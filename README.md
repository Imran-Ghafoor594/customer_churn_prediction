## Project Overview

This project is an end-to-end **Machine Learning system** that analyzes customer behavior data, identifies churn patterns, and predicts which customers are likely to leave. It includes a fully interactive **Streamlit dashboard** for real-time predictions and business insights.

## Setup & Installation

### 1. Clone or download the project

```bash
cd churn_prediction
```

### 2. Create a virtual environment

```bash
python -m venv .venv
```

### 3. Activate the environment

**Windows:**
```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy RemoteSigned
.venv\Scripts\Activate.ps1
```

**Mac / Linux:**
```bash
source .venv/bin/activate
```

### 4. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn scikit-learn joblib streamlit
```

---

##  How to Run

### Step 1 — Run the Notebook

Open `Churn_Prediction.ipynb` in Jupyter or VS Code and run all cells.

This will:
- Load and clean the dataset
- Engineer features
- Train Logistic Regression and Random Forest models
- Save models to the `models/` folder

```bash
jupyter notebook Churn_Prediction.ipynb
```

### Step 2 — Launch the Dashboard

```bash
streamlit run app.py
```

Then open your browser at: `http://localhost:8501`

## Dashboard Pages

| Page | Description |
|------|-------------|
| **Overview** | Total customers, risk distribution, segmentation charts, revenue at risk |
| **Visualizations** | Revenue trends, correlation heatmap, feature importance chart |
| **Predict Customers** | Upload any CSV for bulk predictions OR manually enter a single customer's details |

### CSV Upload (Optional)
- Upload a customer CSV file from the sidebar
- The file **does not need a Churn column** — model will predict it
- Results can be filtered by risk category and downloaded

### Single Customer Prediction
- Works **without any CSV upload**
- Fill in customer details manually
- Get instant churn probability and risk category (Low / Medium / High)

## Models Used

| Model | Accuracy | Precision | Recall | F1 Score | ROC-AUC |
|-------|----------|-----------|--------|----------|---------|
| Logistic Regression | 0.8357 | 0.7033 | 0.5269 | 0.6025 | 0.8517 |
| Random Forest | 0.8180 | 0.6684 | 0.4552 | 0.5416 | 0.8247 |

> **Logistic Regression** performed better on this dataset and is used for dashboard predictions.
